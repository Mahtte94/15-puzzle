### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Mahtte94/15-puzzle.git
   cd puzzle
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run the game**
   ```bash
   # For development
   npm run dev
   
   # Or simply open index.html in your browser
   open index.html
   ```
